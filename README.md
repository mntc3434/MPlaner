# MPlaner 🏋️‍♂️📈

**MPlaner** is a premium, high-tech fitness and muscle-tracking application built with React Native and Expo. Designed for serious trainees, it combines a sleek **Glassmorphism UI** with advanced performance tracking.

## ✨ Key Features

- **🏆 Performance Tracking**: Integrated rest timers, personal record (PR) tracking, and live session progress.
- **🛡️ Elite Training Plan**: Specialized 5-day Upper Body Hypertrophy split (Gym & Dumbbell focused).
- **📊 Advanced Analytics**: Real-time BMI calculation, weight trend analytics, and achievement milestones.
- **🎨 Premium Design**: Dual-theme system (Deep Dark Glass / Pearl Light) with a stunning glassmorphism aesthetic.
- **🍴 Meal Logging**: Integrated nutrition tracker to manage daily calorie targets for mass gain.
- **👤 Custom Profiles**: Fully editable user stats including starting/goal weight, height, and display name.

## 🚀 Tech Stack

- **Framework**: Expo (SDK 52)
- **Engine**: React Native 0.76 (Stable Engine)
- **Navigation**: React Navigation 6
- **Storage**: AsyncStorage
- **Charts**: React Native Chart Kit
- **Design**: Vanilla CSS with Expo Blur (Glassmorphism)

## 🛠️ Getting Started

### Prerequisites
- Node.js & npm
- Expo Go (for testing)
- EAS CLI (for building)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mntc3434/MPlaner.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npx expo start
   ```

### Building for Android
To generate a standalone `.apk`:
```bash
eas build -p android --profile preview
```
